{% comment %} Render a link to a category. If there exists a page with URL /CATEGORY for the given category then render a link to that page, with the category name as the link text. If there's no page for the category then just render the category name (not a link). Usage ===== {% include category_link.html category=some_category %} {% endcomment %} {%- assign category_url = "/" | append: category | append: "/" -%} {%- assign category_page = nil -%} {%- for page in site.pages -%} {% if page.url == category_url %} {% assign category_page = page %} {% break %} {% endif %} {%- endfor -%} {%- if category_page -%} {{ category_page.title }} {%- else -%} “{{ category }}” {%- endif -%}
